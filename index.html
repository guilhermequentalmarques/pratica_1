Praticando com o Git e Github

O que é o GIT?

Git é um repositório local, que armazena nossos arquivos e as versões dos nossos arquivos

O que é o Github

GitHub é um repositório remoto, que também pode ser usado como uma rede social,que você pode colocar seus trablhos, para que os outros possam ver e modifica-los
